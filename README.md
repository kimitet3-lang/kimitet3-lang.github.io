# 🧠 Tet Prompt + Affiliate AI Hub

Welcome to the **Tet Prompt Hub**, a transparent, open-source repository bridging advanced prompt engineering with practical, ethical monetization.

## 🌟 The Three Pillars

1. **The Storefront (`/index.html`):** A clean, fast GitHub Pages site showcasing high-ROI AI tools and accepting community support.
2. **The Engine (`PROMPTS.md`):** A living library of strictly structured, version-controlled AI skills (v1, v2, v3).
3. **The Management (GitHub Projects):** Transparent tracking of every prompt upgrade, bug fix, and new feature.

## 🤖 Powered by a Fused AI Workflow

Our prompts and site are built using a specialized multi-model approach:

* **DeepSeek 4 Pro (The Architect):** Handles complex code generation, HTML/CSS logic, and deep prompt structuring.
* **Muse Spark AI (The Designer):** Handles UI/UX ideation, creative hooks, and visual formatting.
* **Meta AI (The Marketer):** Handles persuasive copywriting, persona testing, and distribution strategy.

## 🚀 Quick Start

1. Go to [`PROMPTS.md`](./PROMPTS.md) to copy our structured master prompts.
2. Visit our [Live Site](https://kimitet3-lang.github.io) to explore recommended tools.
3. Check the **Projects Tab** to see what we are building next or to contribute!

- **Zero Hype:** We do not promise automatic income. We provide tools, skills, and processes.

- **Open Source:** Fork this repo, adapt the prompts, and build your own hub.
