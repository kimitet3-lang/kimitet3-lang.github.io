# Tet Prompt + Affiliate AI Hub – Whitepaper

## 1. Vision

The Tet Prompt + Affiliate AI Hub is a GitHub-based system that turns reusable AI prompts into real, monetizable assets while staying transparent and student-friendly. It combines a public landing page, a structured prompt library, and a project management layer to keep everything versioned, auditable, and easy to learn from.

## 2. Core Pillars

### 2.1 Public Presence (GitHub Pages)

The hub uses `kimitet3-lang.github.io` as a static site that showcases affiliate offers in a clear, non-hype way, provides a PayPal donation entry, and links to the prompt library.

### 2.2 Prompt Engineering Hub (`PROMPTS.md`)

The single source of truth for all AI skills. Each skill follows a strict, reusable schema (Title, Theme, Goal, Workflow, Rules, Examples, Template) with one-shot examples by default.

### 2.3 Evolution & Management (GitHub Projects)

Tracks new prompts (v1), refinements (v2, v3), site changes, and AI platform experiments. Every change is linked to an issue.

## 3. Design Principles

1. **Clarity first:** No confusing funnels. Everything is documented in Markdown.
2. **One-shot by default:** Each prompt includes one clear example.
3. **No "automatic money" claims:** Focuses on learning, tools, and process.
4. **Model portability:** Prompts run on multiple AI platforms with minimal edits.

## 4. AI Platforms Integration

- **DeepSeek 4 Pro:** Advanced reasoning, code generation, and logic structuring.
- **Muse Spark AI:** Creative generation, layout ideas, and hook writing.
- **Meta AI:** General-purpose chat, copywriting, and conversational testing.

## 5. Versioning and Upgrades

Each skill starts at `v1`. Improvements are tagged as `v2`, `v3`, etc., and linked to a GitHub Issue. This creates a public history of prompt evolution.

## 6. Safety and Ethics

- No automatic income promises.
- Clear disclosures for affiliate links and donations.
- Focus on skill-building, not guaranteed outcomes.
